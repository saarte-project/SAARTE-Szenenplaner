Voice Commands of SAARTE-Szenenplaner
====================================

Each voice command also has a keyboards shortcut. This shortcut is documented after the words used in the command in the following.

+ **Show Menu** `Return`
+ **Hide Menu** `Escape`
+ **Move Object** `M`
+ **Rotate Object** `Tabulator`
+ **Show Debug** `A`
+ **Hide Debug** `B`
